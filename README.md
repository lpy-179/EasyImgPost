# EasyImgPost
 to make it easy to upload img and add a watermark on it ,it will auto copy markdown url to your clipboard
 
 # 为对接自己图床提供了有效的途径，
 
 # GUI界面 written by Go
 
 # 特点
 
 - 复制图像到剪贴板，点击copy,会自动添加水印
 - 水印可自己定制，
 - copy后，会自动将图片发送到图床
 - 同时会自动将markdown格式的链接复制到剪贴板，全程只需要点击一下
 - 完全不用担心占用问题
 - 水印图片会同步输出至./out/
